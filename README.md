# yusuf-portfolio

Frontend portfolio skeleton designed for GitHub Pages.

## Tech

- HTML
- CSS
- JavaScript

## Local preview

Open `index.html` directly in a browser.

## Customize quickly

1. Update resume link in `script.js`:

```js
resumeUrl: "https://drive.google.com/file/d/REPLACE_WITH_YOUR_RESUME_ID/view"
```

2. Replace placeholder content:

- `config.skills`
- `config.projects`
- Hero/About/Contact copy in `index.html`

## Deploy on GitHub Pages

1. Push these files to repository `yusuf-portfolio` on branch `main`.
2. In GitHub, open `Settings -> Pages`.
3. Under **Build and deployment**, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Save.
5. Your site will be live at:

```text
https://yusufansari2001.github.io/yusuf-portfolio/
```
